Mac中mysql密码忘记

1.关闭mysql服务

sudo /usr/local/mysql/support-files/mysql.server stop 'your password' # 开机密码

2.进入mysql安装路径

cd /usr/local/mysql/bin/

3.跳过mysql安全认证

 ./mysqld_safe --skip-grant-tables & # 会发现关闭的mysql server状态会变成running

4.进入mysql

./mysql
# 登陆成功
Welcome to the MySQL monitor.  Commands end with ; or \g.
Your MySQL connection id is 12

5.刷新权限

FLUSH PRIVILEGES; # Query OK, 0 rows affected (0.01 sec)

6.修改密码

ALTER USER 'root'@'localhost' IDENTIFIED BY 'your password';

7.退出并验证

quit; # 退出mysql命令
mysql -u root -p # 重新登录
